# DocMind Skill

自动分析项目结构，为每个文件夹和重要文件生成面向小白的说明文档。

## 功能

- 扫描项目所有文件夹
- 为每个文件夹生成说明.md文件
- 为代码文件生成头部注释
- 在根目录生成项目总览README.md和文件关系图.md

## 目标用户

完全不懂代码的小白，需要知道每个文件夹和文件是干什么的。

## 使用方法

在项目中运行：

```
/docmind
```

## 输出结构

```
项目根目录/
├── 项目总览.md          # 整个项目是干啥的
├── 文件关系图.md        # 文件之间的联系
├── src/
│   ├── 说明.md         # 这个文件夹是干啥的
│   └── components/
│       ├── 说明.md
│       └── Button.tsx   # 文件头部会有注释
├── docs/
│   └── 说明.md
└── ...
```

## 技术架构

- 文件扫描器：递归扫描项目目录结构
- 分析器：分析文件内容和用途
- 生成器：生成面向小白的说明文档
- 注释器：为代码文件添加头部注释

## 配置选项

在 skill.yaml 中可以配置：
- `max_depth`: 扫描最大深度（默认5层）
- `exclude_patterns`: 排除的文件/文件夹模式
- `default_output_lang`: 输出语言（默认中文）
